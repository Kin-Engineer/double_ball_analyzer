# 双色球分析器 (Double Ball Analyzer)

## 项目简介

双色球分析器是一个功能强大的Python工具，专门用于分析和预测中国福利彩票双色球的开奖数据。

## 核心特性

### 数据管理
- 自动数据采集：实时爬取最新双色球开奖数据
- 数据清洗：智能清洗和验证原始数据
- 数据库存储：使用SQLite存储历史数据
- 数据同步：定时更新数据源

### 统计分析
- 基础统计：频率、均值、方差等统计指标
- 热冷号分析：识别近期热门号码和冷门号码
- 趋势分析：号码走势和模式识别
- 概率计算：各号码出现概率分析

### 智能预测
- 机器学习模型：集成XGBoost、LightGBM等先进算法
- 多模型融合：结合多个模型的预测结果
- 特征工程：自动提取有效特征
- 模型评估：全面的预测效果评估

### 可视化展示
- 图表生成：自动生成各种分析图表
- 交互界面：用户友好的操作界面
- 报告导出：生成详细的分析报告
- 实时展示：动态展示分析结果

## 快速开始

### 环境要求
- Python 3.8 或更高版本
- pip 包管理工具

### 安装步骤

```bash
# 克隆项目到本地
git clone https://github.com/Kin-Engineer/double_ball_analyzer.git

# 进入项目目录
cd double_ball_analyzer

# 安装依赖包
pip install -r requirements.txt

# 运行主程序
python main.py
```

### 使用说明

1. 首次运行：程序会自动下载历史数据并初始化数据库
2. 数据分析：选择相应的分析功能进行数据探索
3. 模型训练：训练机器学习模型用于预测
4. 结果查看：查看分析报告和可视化图表

## 配置说明

主要配置文件 config.py 包含以下设置：

- 数据库配置：SQLite数据库路径和参数
- 数据源设置：双色球数据源URL
- 分析参数：各种分析算法的参数
- 模型参数：机器学习模型训练参数
- 可视化设置：图表样式和输出格式

## 数据分析方法

### 1. 统计分析方法
- 描述性统计：均值、中位数、标准差
- 频率分析：各号码出现次数统计
- 分布检验：检验号码分布规律

### 2. 热冷号分析
- 热号识别：近期高频出现的号码
- 冷号追踪：长期未出现的号码
- 温号区间：出现频率适中的号码

### 3. 趋势预测
- 时间序列分析：基于历史数据的趋势分析
- 移动平均：平滑数据波动
- 回归分析：预测未来趋势

### 4. 机器学习
- 特征选择：选择对预测有用的特征
- 模型训练：训练多种机器学习模型
- 模型评估：使用交叉验证评估模型性能
